## Installation

1. Clonez ce dépôt

    ```bash
    git clone https://github.com/diamond-diadem/DAE-spectra.git
    cd DAE-spectra
    ```

2. Créez l'environnement Conda

    ```bash
    conda env create -f environment.yml
    conda activate dae-env
    ```
